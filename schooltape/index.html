Schooltape

Schooltape is a chromium extension to customize the look and feel of Schoolbox. It is free and open source and available for download here! Even you can become a developer and help the progress of Schoolbox, all you need is a Github account!


Features:

- Custom themes for Schoolbox such as dark mode, modern (coming soon™️) and insane brightness (coming soon™️)

- Update the greeting message to display the current period.

- Intergrate the current music timetable with normal timetable (coming soon™️)

- Periodic reload (When page is inactive) (coming soon™️)
