Thank you for installing Schooltape!

Usage guide:

- To customise the extension settings right-click on the extension icon in the toolbar and select "Extension Options"

- To enable or disable the extension left-click on the extension icon in the toolbar. The extension icon should now be gray.


Features:

- Custom themes for Schoolbox such as dark mode, modern (coming soon™️) and insane brightness (coming soon™️)

- Update the greeting message to display the current period.

- Intergrate the current music timetable with normal timetable (coming soon™️)

- Periodic reload (When page is inactive) (coming soon™️)
